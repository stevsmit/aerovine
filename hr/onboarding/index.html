    1. HR & Compliance (Day 1 Focus)¶

    Responsible: HR Department

    Status Task Details
    $\Box$ Welcome and Introduction Introduce to key team members, provide office tour, and explain safety protocols.
    $\Box$ I-9 and Tax Forms Complete Form I-9 (Employment Eligibility Verification) and Federal/State W-4 forms.
    $\Box$ Benefits Enrollment Review health/dental/vision plans; assist with 401(k) enrollment forms.
    $\Box$ Policy Review & Signatures (CRITICAL) Obtain signed copies of the following documents:
    $\Box$ Employment Offer Letter & Agreement
    $\Box$ Proprietary Information & Invention Assignment Agreement (PIIAA)
    $\Box$ Non-Compete and Non-Solicitation Agreement
    $\Box$ AeroVine Employee Handbook Acknowledgment
    $\Box$ Background Check Completion Confirm successful conclusion of the required background check.
    $\Box$ System Access Approval Initiate access requests based on job role (e.g., Jira, Accounting Software).

    2. IT & Hardware Setup (Day 1/Week 1)¶

    Responsible: IT or Software Department Lead

    Status Task Details
    $\Box$ Workstation Provided Assign and set up the workstation (Desk, Monitor, Ergonomic accessories).
    $\Box$ Laptop/Computer Issued Issue the primary machine (MacBook Pro or Linux Workstation for Engineers).
    $\Box$ Required Software Installed Ensure core applications are installed: VS Code, LibreOffice, Slack, Video Conferencing client.
    $\Box$ Security Setup Set up VPN access, two-factor authentication (2FA), and issue Security Key/Badge.
    $\Box$ Network Credentials Provide Wi-Fi credentials and credentials for internal servers/network drives.
    $\Box$ Developer Tools (Engineers Only) Install Homebrew (or equivalent package manager), Python 3, Git, and relevant virtualization tools (e.g., Docker/Podman).
    $\Box$ Email & Calendar Access Set up and test Company email and calendar accounts.

    3. Software/Engineering & IP Access (Week 1 Focus)¶

    Responsible: Manager and Senior Engineer/Team Lead

    Status Task Details
    $\Box$ Git & GitHub Access Create account, grant access to the main AeroVine code repositories (read and write).
    $\Box$ Project Management Access Grant access to the project tracking system (e.g., Jira or Trello) and assign initial tickets.
    $\Box$ Cloud Credentials Issue credentials for necessary cloud platforms (e.g., AWS S3 bucket for training data, Azure DevOps, CI/CD pipeline access).
    $\Box$ Code Base Orientation Walk through the structure of the main code repositories, including the core flight control software and the AI/Computer Vision models.
    $\Box$ First Commit Training Guide the new hire through cloning the repository, making a small, non-critical change (e.g., to the README), committing, and pushing the change.
    $\Box$ Local Dev Environment Setup Ensure the employee can successfully run the local drone simulation or a core module of the AI software on their machine.
    $\Box$ Drone Lab Access If role requires, grant access to the physical drone testing lab and explain safety zones/protocols.

    4. Integration & Training (First Month)¶

    Responsible: Manager and Mentor/Buddy

    Status Task Details
    $\Box$ Assign Mentor/Buddy Designate a senior team member to serve as the new hire's go-to resource for questions.
    $\Box$ Department Overview Schedule meetings with leads of interdependent departments (e.g., Hardware, Sales, Finance) to understand their role in the drone lifecycle.
    $\Box$ Mandatory Compliance Training Complete online training modules for:
    $\Box$ Data Security & Privacy
    $\Box$ Export Compliance (ITAR/EAR)
    $\Box$ Anti-Harassment/Code of Conduct
    $\Box$ Initial Goal Setting Define clear, measurable goals (OKRs) for the first 30/60/90 days.
    $\Box$ 30-Day Check-in Manager conducts a formal check-in to assess integration, tool proficiency, and goal progress.

    Manager Confirmation¶

    I confirm that all essential tasks above have been completed or are scheduled and that [New Hire Name] is fully integrated into the team and has access to all necessary resources and documentation.
